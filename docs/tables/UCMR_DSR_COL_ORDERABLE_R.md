# UCMR_DSR_COL_ORDERABLE_R

> This table stores relationships between a sequencing activity type with the list of columns it can have and their sequence.

**Description:** Unified Case Manager Reference - DSR Column and Orderable Relationship  
**Table type:** REFERENCE  
**Primary key:** _(not published — see note)_  
**Columns:** 13
