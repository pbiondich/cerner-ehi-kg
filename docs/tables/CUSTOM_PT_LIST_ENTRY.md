# CUSTOM_PT_LIST_ENTRY

> This table contains the entries of custom patient lists. The list itself is stored on the CUSTOM_PT_LIST table.

**Description:** This table contains the entries of custom patient lists.  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_
