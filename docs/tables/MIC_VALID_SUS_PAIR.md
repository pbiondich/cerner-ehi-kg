# MIC_VALID_SUS_PAIR

> This reference table holds the list of valid pairings for ordering identification and susceptibility tests.

**Description:** Microbiology Valid Susceptibility Pairs  
**Table type:** REFERENCE  
**Primary key:** _(not published — see note)_  
**Columns:** 7

[← index](../index.md)
