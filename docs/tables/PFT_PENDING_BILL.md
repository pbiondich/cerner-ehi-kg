# PFT_PENDING_BILL

> Contains a placeholder for a bill in the system whose metadata has been successfully generated by the billing server but has not been submitted. This type of bill is not visible to users of the system.

**Description:** ProFit Pending Bill  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_
