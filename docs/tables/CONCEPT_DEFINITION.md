# CONCEPT_DEFINITION

> Each row in the Concept Definition table is a definition of the concept. A concept may have definitions contributed from multiple source vocabularies.

**Description:** Concept Definition  
**Table type:** REFERENCE  
**Primary key:** _(not published — see note)_  
**Columns:** 16

[← index](../index.md)
