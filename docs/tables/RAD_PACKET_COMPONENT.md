# RAD_PACKET_COMPONENT

> This table identifies those patient packet components that have printed for a given order.

**Description:** Radiology Packet Component  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_  
**Columns:** 8

[← index](../index.md)

| # | Column | Type | Null? | Flags | Definition |

| 2 | `PRINT_DT_TM` | DATETIME |  |  | The PRINT_DT_TM contains the date and time the packet was printed. |
