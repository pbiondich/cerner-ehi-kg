# ORG_NTWK_PRVDR

> This table stores organizations that are health care providers for a given provider network. It stores begin and end effective dates, and specialty codes for the provider as well.

**Description:** Health Plan Network Providers that are of type "Organization"  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_  
**Columns:** 15

[← index](../index.md)
