# DM_DCM_UOW

> A table to hold the units of work of a work plan for the change management solution

**Description:** Data Change Management Unit of Work  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_  
**Columns:** 16

[← index](../index.md)

| # | Column | Type | Null? | Flags | Definition |
|--:|--------|------|:-----:|-------|------------|

| 2 | `DATA_DETAIL` | VARCHAR(4000) |  |  | Detail of data generated by this unit of work |

| 8 | `START_DT_TM` | DATETIME |  |  | The time the unit of work started |

| 15 | `UPDT_ID` | DOUBLE | NOT NULL |  | The person_id of the person from the personnel table (prsnl) that caused the last insert or update of the row in the table. |
