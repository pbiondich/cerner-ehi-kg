# CR_XML_DOC_REQUEST

> This table stores requests made to the Clinical Documentation Service and indicates when the request was made and whether the XML document generation was successful for a given patient and/or encounter.

**Description:** CR XML Document Request  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_  
**Columns:** 14
