# DM_PROPERTY_SET_CTL

> Table is used to indicate that any type of change has occurred in DM_PROPERTY_SET, DM_OVERIDE_PROPETIES, or DM_MODEL_PROPERTIES. This table is watched by any code that has cached properties so that they know when to refresh them. This table should be updated in the same transaction as the properties being updated to ensure consistency.
