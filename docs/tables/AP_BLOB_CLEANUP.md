# AP_BLOB_CLEANUP

> This table contains a list of all images that can be deleted from the image archive by the ProView image server.

**Description:** ap_blob_cleanup  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_  
**Columns:** 7

[← index](../index.md)

| 2 | `STORAGE_CD` | DOUBLE | NOT NULL |  | Contains the device location where the blob is stored. |
