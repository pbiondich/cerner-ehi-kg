**Description:** This is a many-to-many relationship resolution table.  
**Table type:** REFERENCE  
**Primary key:** _(not published — see note)_  
**Columns:** 7

| # | Column | Type | Null? | Flags | Definition |

| 1 | `STRT_MFG_ID` | DOUBLE | NOT NULL |  | Contains the DBMS assigned unique key field value from strt_mfg table. |
| 2 | `STRT_SUPPLIER_ID` | DOUBLE | NOT NULL |  | Contains the DBMS assigned unique key field value frmo strt_supplier table. |

| 6 | `UPDT_ID` | DOUBLE | NOT NULL |  | The person_id of the person from the personnel table (prsnl) that caused the last insert or update of the row in the table. |
