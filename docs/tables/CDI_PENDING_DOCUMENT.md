# CDI_PENDING_DOCUMENT

> This table represents a CPDI Document within the batch capture process and contains information about the document in the 3rd party imaging system. While a document is pending validation, values shall be as sent from external feeds and may not be consistent with each other (eg, encntr_id or person_id values may be invalid).
