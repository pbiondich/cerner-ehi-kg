# LH_CNT_IRFPAI_ICD

> This table is used to store the sequence of ICD codes for a given encounter. These codes can be freetext as well as having a sequence set. It is possible that a user can start the process, need to save the results, and come back to finish. That is when the data will be retrieved from this table.
