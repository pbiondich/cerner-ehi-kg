# IM_DEVICE_DISCIPLINE_R

> The im_device_discipline_r table describes the discipline(s) for which a device is used. For example, a modality device may be used for radiology purposes only, or it may be used for both radiology and cardiology purposes.

**Description:** Imaging Device Discipline Relationship  
**Table type:** REFERENCE  
**Primary key:** _(not published — see note)_  
**Columns:** 7
