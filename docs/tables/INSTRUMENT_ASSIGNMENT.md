# INSTRUMENT_ASSIGNMENT

> Table used to show the person ID that is currently assigned to an instrument. This is the person ID that will be posted as the perform_personnel_id when results come from an instrument.

**Description:** Instrument Assignment  
**Table type:** ACTIVITY  
**Primary key:** _(not published — see note)_  
**Columns:** 10
